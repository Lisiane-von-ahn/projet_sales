# Supermarket Sales Dashboard

## Overview

This Streamlit-based dashboard provides a detailed analysis of supermarket sales data. The dashboard includes various visualizations such as daily revenue, revenue by product type, revenue by branch, revenue by payment type, and branch ratings.

## Features

- **Daily Revenue**: Visualizes the daily revenue for the selected month.
- **Revenue by Product Type**: Shows the revenue distribution across different product lines.
- **Revenue by Branch**: Summarizes the total revenue generated by each branch.
- **Revenue by Payment Type**: Illustrates the revenue contribution by each payment method.
- **Branch Ratings**: Displays the average customer ratings for each branch.

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/supermarket-sales-dashboard.git
    ```

2. Navigate to the project directory:
    ```bash
    cd supermarket-sales-dashboard
    ```

3. Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the Streamlit app:
    ```bash
    streamlit run app.py
    ```
